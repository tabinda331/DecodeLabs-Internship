# 🤖 Task 1 — Rule-Based AI Chatbot

## 📌 Overview

This project is an advanced rule-based AI chatbot developed as part of the DecodeLabs Internship program.

The chatbot simulates intelligent interaction using:

* control flow
* if-else logic
* intent recognition
* conversational memory
* typo correction
* NLP preprocessing

---

## 🚀 Features

* Greeting recognition
* Exit commands
* Continuous conversation loop
* Intent detection
* Name memory
* Typo correction
* Rule-based responses
* Interactive chatbot UI using Gradio

---

## 🛠 Technologies Used

* Python
* Gradio
* Regex
* Difflib
* Datetime

---

## ▶️ Run the Project

Install dependencies:

```bash
pip install gradio
```

Run the chatbot:

```bash
python app.py
```
